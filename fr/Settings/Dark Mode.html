Mode sombre

Oak Diary supporte un mode Dark très esthétique. Toutes les pages du journal de bord supportent le mode sombre.

  • Activer le mode Dark en cliquant sur l'interrupteur
  • Ajoutez une période de temps pour activer automatiquement le mode Dark ou vous pouvez le faire en vous basant sur le système
